**Salary Prediction using Linear Regression**



**Project Overview**



This project develops a regression model to predict employee salaries using Linear Regression.



The project demonstrates data preprocessing, exploratory analysis, model training, prediction, and evaluation workflows.



**Technologies Used**



\* Python

\* Pandas

\* NumPy

\* Matplotlib

\* Scikit-Learn



**Algorithm Used**



\* Linear Regression



**Workflow**



1\. Data Loading

2\. Data Exploration

3\. Data Cleaning

4\. Train-Test Split

5\. Model Training

6\. Prediction

7\. Performance Evaluation



**Evaluation Metrics**



\* Mean Absolute Error (MAE)

\* Mean Squared Error (MSE)

\* R² Score



&#x20;**Business Goal**



Develop a predictive system to estimate salaries using regression analysis techniques.



**Results**



Successfully built a Linear Regression model for salary prediction using analytical preprocessing and model evaluation techniques.
